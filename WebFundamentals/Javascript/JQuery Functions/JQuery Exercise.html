.click

Click is an event.

Hide / Show

.hide will hide the items in this class

.show will show or unhide the items in this class

.toggle

toggle will hide and unhide items with a designated speed that you give it, like "slow" or "fast". Also, you dont need a second button or JQuery code to unhide the toggle.

$("#toggle").click(function(){ $(".block-toggle p").toggle(); });

slideUp / slideDown

Slide up will hide images using a sliding motion. You can designate the speed of the slide within the slideUp parameter. eg: .slideUp("slow")

Slide Down with unhide images that you've slid up

slideToggle

toggle will hide and unhide items with a designated speed that you give it, like "slow" or "fast". Slide Toggle uses only one button, unlike the Toggle Feature

FadeIn / FadeOut

Fade out will fade the element youve selected

fade in will fade the faded out image back in

Add Class

Adding a class will add a class that you've designated to style or color. In this case, the paragraph tag was changed to green and underlined.

Before / After / Append / Prepend

Before will add text or an attribute before the selected tag

After will add text or an attribue after the selected tag

Append will add language within the selected element, but after the text or attribute found in that element

Prepend will add language within the selected element, but before the text or attribute found in that element

HTML

HTML lets you create new HTML code within JQuery. It's different from .text because it is interactive, whereas text will return exactly what was written.

Attribute

JQuery attribute allows you to change the CSS attributes to a tag. In this example, the picture width changed.

$("#attr").click(function(){ $(".attribute img").attr('width', "500px", "height", "1000px"); $(".attribute img").attr('height', "50px"); })

Value

Value changes whats in an input:

Text

Text inserts verbatim what is written in the parameters section. Unlike HTML, it cannot act on html tags written in here

Data

Data Jquery transfers information when you click one button, to another location in the HTML. You set what the message says within the parameters of the transferring button.
